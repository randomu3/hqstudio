# Требования: Desktop CRM для HQ Studio

## Контекст
WPF приложение для внутреннего использования сотрудниками автостудии. Работает с общим API.

## Функциональные требования

### FR-1: Авторизация
- Вход по логину/паролю через API
- Сохранение сессии
- Автоматический logout при истечении токена
- Обязательная смена пароля при первом входе

### FR-2: Управление заказами
- Список заказов с фильтрацией по статусу и дате
- Создание/редактирование заказов
- Печать заказа
- Экспорт в Excel

### FR-3: Управление клиентами
- Поиск клиентов
- Создание/редактирование клиентов
- История заказов клиента

### FR-4: Заявки с сайта
- Просмотр новых заявок
- Создание клиента/заказа из заявки
- Изменение статуса

### FR-5: Управление сотрудниками
- Список сотрудников с онлайн-статусом
- Сброс пароля сотрудника (Admin)
- Редактирование профиля

### FR-6: Dashboard
- Статистика за период
- График выручки
- Популярные услуги

### FR-7: Темы оформления
- Тёмная тема (по умолчанию)
- Светлая тема
- Переключение в настройках

## Нефункциональные требования

### NFR-1: Offline режим
- Кеширование данных
- Индикатор отсутствия связи с API

### NFR-2: Производительность
- Пагинация списков
- Rate limiting API запросов
- Кеширование на 5 минут

### NFR-3: UX
- Кастомные диалоги вместо MessageBox
- Анимации загрузки
- Горячие клавиши
