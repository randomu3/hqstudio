# Требования: Админ-панель HQ Studio

## Контекст
Админ-панель для управления контентом сайта, заявками с сайта и клиентской базой автостудии HQ Studio.

## Функциональные требования

### FR-1: Управление заявками (Callbacks)
- Просмотр списка заявок с сайта
- Фильтрация по статусу (новая, в работе, завершена)
- Изменение статуса заявки
- Создание клиента/заказа из заявки

### FR-2: Управление клиентами
- CRUD операции для клиентов
- Поиск по имени, телефону, email
- История заказов клиента

### FR-3: Управление заказами
- Создание заказа с привязкой к клиенту
- Выбор услуг из каталога
- Статусы: новый, в работе, завершён
- Soft delete (архивирование)

### FR-4: Управление услугами
- CRUD для каталога услуг
- Категории услуг
- Цены и описания

### FR-5: Управление контентом сайта
- Редактирование секций главной страницы
- Управление FAQ
- Управление отзывами

### FR-6: Аналитика (Dashboard)
- Количество заявок за период
- Выручка за период
- Популярные услуги

## Нефункциональные требования

### NFR-1: Авторизация
- JWT аутентификация
- Роли: Admin, Editor, Manager
- Разграничение доступа по ролям

### NFR-2: Производительность
- Пагинация для списков
- Кеширование на клиенте
